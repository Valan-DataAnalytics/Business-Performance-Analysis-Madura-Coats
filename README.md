Madura Coats: Business Performance Analysis
Project Overview
This project analyzes the Ambasamudram Branch of Madura Coats Private Limited. It explores the gap between financial stability and operational underutilization.

Key Insights
* Revenue Stability: Reported operating revenue of ₹1,664 crore for FY2023. * Operational Gap: Found a 25% underutilization gap (16M Kg target vs 12M Kg actual production). * Efficiency Risk: Operating at only 60-65% capacity despite a strong net worth of ₹367 crore. * Strategic Drags: 75% of identified issues, such as water transport costs and export rule violations, are categorized as High Impact.

Tools Used
* Microsoft Excel: Data preparation and trend calculations. * Power BI: Executive-level visualization and KPI summaries
